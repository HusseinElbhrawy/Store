# Store App 

A new Flutter project ,it is an e-commerce app with firebase backend  

### Features 
1. Native Splash Screen
2. Login with Email and Password
3. Login With Google and Facebook Account
4. Login as a guest 
5. Reset password 
6. Integration with Payment gateway
